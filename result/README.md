# 结果包

本目录包含用于公开发布的精简实验结果。

## r-RLZSA

`r-RLZSA/` 包含创新点 1 在七个数据集上的结果表：

- `combined_build_summary.csv`
- `combined_final_test_performance.csv`
- `combined_validation_theta_scan.csv`
- `combined_workload_occ_stats.csv`
- 各数据集目录下的单数据集摘要
- `all_7_datasets_statistics.md`

## Vr-RLZSA

`Vr-RLZSA/` 包含创新点 2 的结果表：

- `varint_v2_cross_dataset/`：Move-r、Move-r-RLZ、Move-r-RLZEnd 与
  Vr-RLZSA 的对比。
- `vrrlzsa_ablation/`：r-RLZSA raw fields、Vr-RLZSA-v1 与 Vr-RLZSA-v2 的对比。
- `field_frequency_top100/`：用于绘图和论文分析的精简 SR/CPL top-100 排名频率数据。

完整本地日志和生成的索引未包含在内。
